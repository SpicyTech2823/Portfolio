# Personal Portfolio Website

A modern and responsive personal portfolio website built with React.js and Tailwind CSS.  
This project showcases my skills, projects, education, experience, and contact information in a clean and professional design.

---

## 🚀 Features

- Responsive design for mobile, tablet, and desktop
- Modern UI using Tailwind CSS
- Fast performance with React.js
- Smooth navigation
- Project showcase section
- Skills section
- About me section
- Contact form / contact information
- Easy customization

---

## 🛠️ Technologies Used

- React.js
- Tailwind CSS
- JavaScript
- HTML5
- CSS3
- Vite / Create React App
